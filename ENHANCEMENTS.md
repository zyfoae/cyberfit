# CyberFit Enhancement Ideas

## Immediate Enhancements (Next Sprint)

1. **Dark Mode Toggle**
   - Add theme switch in Profile/Settings
   - Persist theme preference in AsyncStorage
   - Smooth transition between themes

2. **Barcode Food Scanner**
   - Integrate expo-barcode-scanner
   - Add scan button in Quick Add food modal
   - Connect to nutrition database API (conceptual)

3. **Workout Timer**
   - Add countdown timer for rest periods
   - Visual and audio cues
   - Customizable intervals

4. **Progress Photos**
   - Add photo upload in Profile
   - Display progress grid
   - Before/after slider

5. **Custom Workout Builder**
   - Drag-and-drop exercise arrangement
   - Save as reusable templates
   - Share workout routines

6. **Nutrition Goals**
   - Macronutrient tracking (carbs/protein/fat)
   - Micronutrient tracking (vitamins/minerals)
   - Water intake goals by activity level

7. **Social Features**
   - Friend system
   - Activity feed
   - Commenting on workouts/meals
   - Achievement sharing

8. **Challenges System**
   - Daily/weekly challenges
   - Streak tracking
   - Leaderboards
   - Reward badges

9. **Voice Commands**
   - "Log 250ml water"
   - "Add 300 calorie breakfast"
   - "Start workout timer"

10. **Export & Backup**
    - Export data as JSON/CSV
    - Google Drive/Dropbox backup
    - Email reports
